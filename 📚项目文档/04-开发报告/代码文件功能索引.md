# 山东非遗可视化系统 - 代码文件功能索引

为了帮助团队成员快速理解系统架构，下表总结了现有核心源文件的功能职责：

## 1. 核心窗体与集成 (Core & Integration)
| 文件名 | 所属角色 | 核心职责简述 |
| :--- | :--- | :--- |
| `Form1.cs` | Member A | **系统枢纽**。初始化助手类、绑定基础事件、管理工具切换逻辑。 |
| `Form1.Data.cs` | Member D | **数据中台**。提供面向统计查询的 API，如各市数据计数、分类统计。 |
| `Form1.Visual.cs` | Member E | **可视化演示**。管理演示模式 UI、图层渲染顺序优化、全屏演示逻辑。 |
| `Form1.Tools.cs` | Member C | **业务工具箱**。处理 MXD/SHP 加载、坐标点数据导入、空间分析触发。 |
| `Form1.Editor.cs` | Member B | **编辑与触发器**。集成图表看板初始化、编辑工具触发、测量工具触发。 |

## 2. 逻辑助手类 (Logic Helpers)
| 文件名 | 核心职责简述 |
| :--- | :--- |
| `DataHelper.cs` | 底层数据操作：GDB 创建、全国数据瘦身（提取山东）、重叠点位离散化。 |
| `EditorHelper.cs` | 编辑事务管理：开始/停止编辑、点线面要素交互创建、动作撤销与保存。 |
| `MeasureHelper.cs` | 交互式测量：实时计算并反馈地图上的距离、周长与面积。 |
| `LayoutHelper.cs` | 制图与布局：数据视图向布局视图同步、添加指北针、比例尺与图例。 |
| `UIHelper.cs` | 通用 UI 逻辑：地图导出为图片 (JPG/PNG/BMP)。 |

## 3. 功能性子窗体 (Sub-Forms)
| 文件名 | 功能描述 |
| :--- | :--- |
| `FormChart.cs` | 数据看板，通过图表展示统计分析结果。 |
| `FormAddXYData.cs` | 引导用户将 Excel/坐标点数据转换为 GIS 图层。 |
| `FormBuffer.cs` | 缓冲区分析配置界面。 |
| `FormOverlay.cs` | 叠置分析（相交、擦除等）配置界面。 |
| `FormSymbolize.cs` | 要素符号化界面，如唯一值符号化、分级符号化。 |
| `FormSelectByLocation.cs` | 空间查询（位置查询）配置界面。 |
| `DataoSourceForm.cs` | 属性数据查询与检索界面。 |

---
**提示**：所有代码修改请遵循 `[Member ROLE] Modified: ...` 注释协议。
